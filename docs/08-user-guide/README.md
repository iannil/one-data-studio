## 用户角色

| 角色 | 职责 | 主要功能 |
|------|------|----------|
| 系统管理员 | 平台管理 | 用户管理、数据源配置、系统监控 |
| 数据管理员 | 数据治理 | 元数据扫描、敏感性分类、质量管理 |
| 数据工程师 | 数据开发 | ETL 流程、数据集成、特征工程 |
| 算法工程师 | 模型开发 | 模型训练、部署、监控 |
| 业务用户 | 数据消费 | 智能查询、报表分析、知识库问答 |
