## 基本原则

### 1. 语言规范

- **文档语言**：使用中文
- **代码语言**：使用英文
- **命名规范**：文件名使用英文小写和连字符

### 2. 格式规范

- **文件格式**：Markdown (.md)
- **图表格式**：Mermaid 语法
- **代码块**：使用围栏式代码块，标注语言

### 3. 结构规范

- 每个文档必须有一级标题
- 重要文档需要更新记录
- 配置和示例代码需要注释

### 4. 存放位置

| 文档类型 | 存放位置 |
|----------|----------|
| 进行中的修改 | `/docs/progress/` |
| 已完成的修改 | `/docs/reports/completed/` |
| 验收报告 | `/docs/reports/` |
| 文档模板 | `/docs/templates/` |
| 文档标准 | `/docs/standards/` |
