# 进行中的修改

> 本目录存放**未完成**的修改文档。

## 说明

根据 `CLAUDE.md` 的约定，每次修改都必须延续上一次的进展，修改过程中的进度应随时保存在此目录下。

## 文件命名规范

- 格式：`{功能模块}-{简要描述}.md`
- 示例：`agent-api-向量检索实现.md`、`web-聊天历史功能.md`

## 文档模板

请参考 `/docs/templates/progress-template.md` 模板编写进度文档。

## 完成后的处理

当修改完成并验收通过后，应将文档移动到 `/docs/reports/completed/` 目录。
